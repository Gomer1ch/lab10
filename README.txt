Завдання:
Створити базу даних, що складається з двох взаємопов'язаних 
таблиць. Таблиця «Товари» містить наступні поля: назва, тип, ціна, номер, 
розмір/фасування, час зберігання (до певної дати).
Таблиця «Закази» містить наступні поля: номер заказу, прізвище 
клієнта, ім'я, телефон, номер товару, назва товару, кількість.
База даних повинна бути у вигляді таблиці та у вигляді форми. 
Кількість записів в таблицях не менше 15.
Передбачити можливість додавання, видалення, редагування записів 
у таблицях.
Створити наступні запити:
1. З’ясувати, які товари замовив конкретний клієнт (за номером телефону). 
Вивести поля: прізвище, ім’я, номер товару, назва товару.
2. Знайти усі товари, ціна яких дорівнює введеному числу. Вивести поля: 
назва товару, номер товару.
3. Знайти конкретний товар (за номером) конкретного розміру/фасування 
(введений). Вивести поля: назва товару, номер товару, ціна.
4. Знайти усі товари, час зберігання яких вже сплив. Вивести поля: номер 
товару, назва товару, час зберігання.
5. Обчислити вартість заказу конкретного клієнта (за прізвищем). Товарів 
більше одного. Вивести поля: прізвище клієнта, номер телефону, назва 
товару, ціна, вартість (додати).
Якщо результатів запиту немає, вивести відповідне повідомлення.
